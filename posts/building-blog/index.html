Build Your Blog Using Hugo and Github Page

#Before you start

  1. All hugo commands should be run in the root level
  2. Git commands should be run in the public folder

#Step 1: Install hugo

For Mac user:

brew install hugo

#Step 2: Create a new site

hugo new site myblog

#Step 3: Add a theme

You can find Hugo themes Here

In this case I use meme theme as my example

git clone https://github.com/reuixiy/hugo-theme-meme.git themes/meme

#Step 4: Copy config.toml content from example site to your config.tmol

You can make your own changes in config.toml file

#Step 5: Add some content

hugo new posts/my-first-post.md

Set Draft = false

Command above will generate a my-first-post.md file under the path: ./content/posts

You should always use hugo new posts/file.md to create new files

#Step 6: Deploy to github pageï¼š

  1. Create a new repo, name the repo as: your-github-username.github.io
  2. Generate public folder
hugo --theme=meme --baseUrl="https://your-github-username.github.io" --buildDrafts

After you run the command above, you will see a public folder is created in your blog folder

  1. Deploy Hugo to github
cd public
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/your-github-username/your-github-username.github.io.git
git push -u origin main
  1. Update to github

When you make changes in your blog folder

Run :

hugo

Above code will update corresponding changes to public folder

Then run:

$ cd public
$ git add .
$ git commit -m "add blog post"
$ git push

You can use git status to check your changes

$ git status

#Build the website locally

hugo server -D
